# Row

Rows and cols are a system grid. They are most used to organize content.

## Element

```html
<div class="row">
  <div class="col">...</div>
</div>
```

## Most used helpers

**Sizes**

s1...s12, m1...m12, l1...l12

**Spaces**
no-space, space, small-space, medium-space, large-space

## Example

This will render one or more lines, depends the user screen.

```html
<div class="grid">
  <div class="s12 m6 l3">
    <h5>First</h5>
  </div>
  <div class="s12 m6 l3">
    <h5>Second</h5>
  </div>
  <div class="s12 m6 l3">
    <h5>Third</h5>
  </div>
</div>
```
