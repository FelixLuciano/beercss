# Radio

Radio buttons allow users to select one option from a set.

## Element

```html
<label class="radio">
  <input type="radio">
</label>
```

## Example

```html
<nav>
  <label class="radio">
    <input type="radio">
  </label>
  <label class="radio">
    <input type="radio">
  </label>
  <label class="radio">
    <input type="radio">
  </label>
</nav>

<nav>
  <label class="radio">
    <input type="radio">
    <span>Radio 1</span>
  </label>
  <label class="radio">
    <input type="radio">
    <span>Radio 2</span>
  </label>
  <label class="radio">
    <input type="radio">
    <span>Radio 3</span>
  </label>
</nav>
```
