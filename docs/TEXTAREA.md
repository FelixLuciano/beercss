# Textarea

Textarea fields let users enter and edit long text.

## Element

```html
<div class="field textarea">
  <textarea></textarea>
</div>
```

## Most used helpers

**Forms**

label, textarea, border, round, fill, prefix, suffix

**Sizes**

small, medium, large, extra

**Triggers**

active

## Example

```html
<div class="field textarea label border">
  <textarea></textarea>
  <label>Label</label>
</div>
```

## Triggers 

#### To up/down a label

#### Method 1

Add/remove `active` class on label.

```html
<div class="field textarea label border">
  <textarea></textarea>
  <label class="active">Label</label>
</div>
```

#### Method 2

Call `ui()` after html output.

```html
<div class="field textarea label border">
  <textarea></textarea>
  <label>Label</label>
</div>
```

```js
ui();
```
